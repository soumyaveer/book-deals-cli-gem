# BookDeals

Displays the current **Goodreads** ebook deals.

## Installation

Add this line to your application's Gemfile:

```ruby

gem 'book_deals'

```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install book_deals

## Usage

Type the gem name in Command Line User Interface to execute.

```ruby

$ book_deals

```
