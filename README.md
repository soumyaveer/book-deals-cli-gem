# BookDeals

Displays the book deals from Goodreads website.

## Installation

Add this line to your application's Gemfile:

```ruby

gem 'book_deals'

```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install book_deals

## Usage

To execute in Command Line User Interface, enter the command:

```ruby

$ book-deals

```
